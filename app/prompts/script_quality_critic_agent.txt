━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
When scoring hindi_naturalness, check for these known premium pitfalls.
Any of these present in the script lowers hindi_naturalness automatically.

• "सांस" (wrong) → should be "साँस" (chandrabindu)
• "जो [X] को देखा था" relative clause → should be "जिस [X] ने ... देखा था"
• "[X] की जाल" → should be "[X] का जाल" (जाल is masculine)
• "था। जिसे" or "है। जिसे" (sentence fragment) → should use comma, not full stop
• "तीन-साढ़े-तीन साल बाद" → prefer "लगभग साढ़े तीन साल बाद"
• "बाल-हत्या" in narration → prefer "बच्चे की हत्या"
• "Canada के" or "Canadian" as adjective at hinglish_level 1–2 → prefer "कनाडा के" / "कनाडाई"
• "यह सिर्फ़" repeated more than twice

Add each violation found as a chunk_repair_target with issue_type="hindi_naturalness".
